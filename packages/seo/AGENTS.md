# `@workspace/seo`

**Audience:** agents / LLM.

## Filename suffixes

This package is mostly **plain `*.ts`** (canonical URLs, JSON-LD, alternates).
**If** you add `src/components/` or `src/hooks/` for SEO-related UI or hooks,
follow
[`docs/standards/package-file-suffixes.md`](../../docs/standards/package-file-suffixes.md)
(`*.component.tsx`, `*.hook.ts`, etc.) — the same CI rules apply as for `brand`
/ `core` / `forms`.
